# CourtTrack AI: NyayaLens

## Overview

CourtTrack AI: NyayaLens is a case-centric legal intelligence system that converts complex court judgment PDFs into clear, actionable insights for government officers and citizens.

Instead of reading lengthy legal documents, users can quickly understand:

* What the case is about
* What actions are required
* Who is responsible
* Deadlines and risks involved
* What happens if no action is taken

---

## Problem

Court judgments are:

* Long and difficult to interpret
* Written in complex legal language
* Lacking clear action points

This leads to:

* Missed deadlines
* Poor coordination between departments
* Risk of contempt of court
* Lack of clarity for citizens

---

## Solution

This system transforms:

**Judgment PDF → Structured Action Plan**

Key outputs:

* Case summary (simple language)
* Required actions
* Responsible department
* Deadlines
* Risk level
* Consequences

---

## Core Features

* PDF Upload & Processing
* AI-based Extraction
* Action & Deadline Identification
* Risk Detection (Contempt Risk)
* Human Verification System
* Case-Centric Dashboard
* Citizen-Friendly Explanation

---

## Tech Stack

* Frontend: Next.js + Tailwind
* Backend: FastAPI
* AI: OpenAI / Claude API
* OCR: Tesseract / Vision API
* Database: PostgreSQL / SQLite

---

## Goal

To build a system that ensures no important court directive is missed and every stakeholder clearly understands what needs to be done next.
